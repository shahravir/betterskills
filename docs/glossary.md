# Glossary

| Term | Meaning |
|------|---------|
| **Agent skill** | A folder with `SKILL.md` (and optional files) that instructs an AI agent how to perform a task. |
| **Skill catalogue** | This repository — a curated, versioned collection of portable skills. |
| **Manifest** | `skill.manifest.yaml` — metadata for discovery, install paths, and taxonomy. |
| **Frontmatter** | YAML block at the top of `SKILL.md` (`name`, `description`, …). |
| **Platform** | An IDE or tool that can load skills (Cursor, Claude Code, Windsurf, …). |
| **Kind** | `engineering`, `non-engineering`, or `hybrid` — audience hint, not a quality label. |
| **Vibe coding** | Building software through conversational iteration with an AI pair programmer. |
