# Professional Email

## Gather (ask if missing)

- Recipient relationship (peer, manager, client, external)
- Goal (inform, request, apologize, escalate)
- Desired response and deadline
- Tone preference (neutral, warm, firm)

## Structure

1. **Subject** — Specific, ≤60 chars when possible
2. **Opening** — Purpose in first sentence
3. **Body** — Context → ask → rationale (short paragraphs)
4. **Close** — Clear CTA + thanks/sign-off

## Tone matrix

| Situation | Tone |
|-----------|------|
| Routine update | Neutral, concise |
| Request | Polite, specific deadline |
| Delay/bad news | Direct, accountable, next step |
| Escalation | Factual, no blame language |

## Template

```
Subject: [Specific subject]

Hi [Name],

[One-sentence purpose.]

[2–4 short paragraphs with bullets if listing items.]

[Clear ask: "Could you confirm by Friday?"]

Thanks,
[Name]
```

## Rules

- No passive-aggressive phrasing
- One main ask per email when possible
- Avoid jargon with external recipients
- Do not send on user's behalf — output draft only unless tool can send
